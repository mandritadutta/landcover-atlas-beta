India's Changing Land Use

This dashboard showcases land cover changes across India using satellite-derived classified maps from 2020 and 2024. Understand the impacts of urban growth, deforestation, and agriculture.

Land Cover: 2020 vs 2024

Tools & Methods

This project is a prototype public dashboard built with HTML and CSS. Classified maps are assumed outputs from a machine learning model trained on Sentinel-2 imagery using Python, Rasterio, and Scikit-learn.
